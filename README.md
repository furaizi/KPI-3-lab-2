# Проект: Обчислення постфіксних виразів
Клонуйте репозиторій:
   ```bash
   git clone https://github.com/furaizi/KPI-3-lab-2.git
   cd KPI-3-lab-2
   ```
Встановіть залежності:
```bash
go mod tidy
   ```
Запустіть програму:
```bash
go run cmd/example/main.go -e "1 2 +"
   ```
